## Usage

```html preview=true
<ol class="Breadcrumbs">
  <li class="Breadcrumbs-item"><a href="#">Item 1</a></li>
  <li class="Breadcrumbs-item"><a href="#">Item 2</a></li>
  <li class="Breadcrumbs-item"><a href="#">Item 3</a></li>
</ol>
```

## Active Item

To apply active state to one of the breadcrumb item, use `.is-active` class. Usually, it is added
for the backmost breadcrumb item which usually indicates the current page.

```html preview=true
<ol class="Breadcrumbs">
  <li class="Breadcrumbs-item"><a href="#">Item 1</a></li>
  <li class="Breadcrumbs-item"><a href="#">Item 2</a></li>
  <li class="Breadcrumbs-item is-active">Active item</li>
</ol>
```
