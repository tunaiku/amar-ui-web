---
path: '/components/table'
title: 'Table'
description: 'Displays sets of data across rows and columns.'
---

## Usage

```html preview=true
<table class="Table">
  <thead class="Table-head">
    <tr>
      <th>No</th>
      <th>Request ID</th>
      <th>Request Type</th>
      <th>Date</th>
    </tr>
  </thead>
  <tbody class="Table-body">
    <tr>
      <td>1</td>
      <td>John Doe</td>
      <td>Change Phone Number</td>
      <td>3/3/2020</td>
    </tr>
    <tr>
      <td>2</td>
      <td>Penny Tool</td>
      <td>New Registration</td>
      <td>28/2/2020</td>
    </tr>
    <tr>
      <td>3</td>
      <td>Bodrum Salvador</td>
      <td>Change Phone Number</td>
      <td>24/2/2020</td>
    </tr>
    <tr>
      <td>4</td>
      <td>Dianne Ameter</td>
      <td>Change Phone Number</td>
      <td>23/2/2020</td>
    </tr>
  </tbody>
</table>
```

## Sizes

#### Compact

Apply `.Table--compact` class after `.Table` to make the spacing on each cell smaller, making the
whole table feels more compact.

```html preview=true
<table class="Table Table--compact">
  <thead class="Table-head">
    <tr>
      <th>No</th>
      <th>Request ID</th>
      <th>Request Type</th>
      <th>Date</th>
    </tr>
  </thead>
```
