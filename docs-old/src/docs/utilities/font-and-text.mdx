## Font sizes

```html preview=true
<p class="FontSize-large">Large</p>
<p class="FontSize-xlarge">Extra large</p>
<p class="FontSize-2xlarge">2x Extra large</p>
```

## Text alignment

```html preview=true
<p class="TextAlign-left MarginBottom">
  <strong>This is an example of a left-aligned paragraph.</strong>
  Aute ullamco occaecat ipsum officia eiusmod. Amet exercitation laboris aliquip proident et
  excepteur in proident.
</p>
<p class="TextAlign-center MarginBottom">
  <strong>This is an example of a center-aligned paragraph.</strong>
  Aute ullamco occaecat ipsum officia eiusmod. Amet exercitation laboris aliquip proident et
  excepteur in proident.
</p>
<p class="TextAlign-right MarginBottom">
  <strong>This is an example of a right-aligned paragraph.</strong>
  Aute ullamco occaecat ipsum officia eiusmod. Amet exercitation laboris aliquip proident et
  excepteur in proident.
</p>
<p class="TextAlign-justify">
  <strong>This is an example of a justified paragraph.</strong>
  Aute ullamco occaecat ipsum officia eiusmod. Amet exercitation laboris aliquip proident et
  excepteur in proident.
</p>
```
