---

title: 'Alert Box'
description:
  'Displays short and important message that user needs to know without much interuption.'
---

## Usage

```html preview=true size=compact
<div class="AlertBox AlertBox--danger">
  <div class="AlertBox-text">Lorem ipsum dolor sit amet.</div>
  <div class="AlertBox-action">Action</div>
</div>
```
