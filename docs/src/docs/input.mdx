## Usage

Append `.Input` class on top of a standard `<input>` element to transform it into Amar UI's input
component.

```html preview=true size=compact
<input type="text" class="Input" placeholder="A standard input" />
```

## Themes

By default, an input component doesn't apply any themes out-of-the-box, but you can easily apply
theme as needed by appending the `*--<theme>` modifier class.

```html preview=true size=compact
<input type="text" class="Input Input--primary MarginBottom" placeholder="Primary theme" />
<input type="text" class="Input Input--secondary MarginBottom" placeholder="Secondary theme" />
<input type="text" class="Input Input--danger MarginBottom" placeholder="Danger theme" />
<input type="text" class="Input Input--warning MarginBottom" placeholder="Warning theme" />
<input type="text" class="Input Input--info" placeholder="Info theme" />
```

## Multiline

You can use `textarea` to create a multiline input.

```html preview=true size=compact
<textarea class="Input" placeholder="I'm a textarea input"></textarea>
```

You can apply themes as usual as well.

```html preview size=compact
<textarea class="Input Input--danger MarginBottom"></textarea>
<textarea class="Input Input--success MarginBottom"></textarea>
<textarea class="Input Input--warning MarginBottom"></textarea>
<textarea class="Input Input--info"></textarea>
```
