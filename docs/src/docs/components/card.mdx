## Usage

`.Card` component consists of several elements that you can mix and match according to your needs.

- `.Card`: Card component outer container.
- `.Card-header`: Wrapper for the title and secondary title of the card.
- `.Card-content`: Primary content of the card.
- `.Card-footer`: Wrapper for the call-to-action elements of the card (e.g button, link)

```html preview=true bgColor=lighter size=compact
<div class="Card">
  <div class="Card-header">
    <h5 class="Typography-heading20">Card title here</h5>
    <p class="Typography-regular14 Color-neutral-80">Secondary text here</p>
  </div>
  <div class="Card-content">
    <p class="Typography-regular14 Color-neutral-80">
      Description text here. Contains various supporting text or content.
    </p>
  </div>
  <div class="Card-footer">
    <button class="Button Button--primary Button--fullWidth">Button</button>
  </div>
</div>
```

## Modifiers

#### Elevated card

Append `.Card--elevated` class to add the elevation effect to a card component to make it appear
floated.

```html preview=true bgColor=lighter size=compact
<div class="Card Card--elevated">
  <div class="Card-header">
    <h5 class="Typography-heading20">Card title here</h5>
    <p class="Typography-regular14 Color-neutral-80">Secondary text here</p>
  </div>
  <div class="Card-content">
    <p class="Typography-regular14 Color-neutral-80">
      Description text here. Contains various supporting text or content.
    </p>
  </div>
  <div class="Card-footer">
    <button class="Button Button--primary Button--fullWidth">Button</button>
  </div>
</div>
```
