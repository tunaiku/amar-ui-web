## Usage

To create a select component, simply create an outer element with the class `.Select` followed by
the `<select>` element.

```html preview=true
<div class="Select">
  <select>
    <option value="Jakarta">Jakarta</option>
    <option value="Bandung">Bandung</option>
    <option value="Surabaya">Surabaya</option>
  </select>
</div>
```
