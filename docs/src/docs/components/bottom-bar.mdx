## Usage

To implement a bottom bar, add `.BottomBar` class to transform it into a bottom bar wrapper. Inside
the bottom bar wrapper, `.BottomBar-menu` and `.BottomBar-menuItem` can be used.

- `BottomBar-menu` creates another layer of wrapper to control the menu items inside. Applies `flex` under the hood
  to align menu items horizontally.
- `BottomBar-menuItem` transforms an element into a menu item that organizes the placements of its insides; the menu icon and label.

```html preview=true bgColor=lighter center=true
<div class="BottomBar">
  <ul class="BottomBar-menu">
    <li class="BottomBar-menuItem">
      <a href="#">Pinjamanku</a>
    </li>
    <li class="BottomBar-menuItem">
      <a href="#">Riwayat</a>
    </li>
    <li class="BottomBar-menuItem">
      <a href="#">Bantuan</a>
    </li>
    <li class="BottomBar-menuItem">
      <a href="#">Lainnya</a>
    </li>
  </ul>
</div>
```

## Active State

In order to activate the active state of a bottom bar item, simply append `.is-active` state inside
of `BottomBar-menuItem`.

```html preview=true bgColor=lighter center=true
<div class="BottomBar">
  <ul class="BottomBar-menu">
    <li class="BottomBar-menuItem">
      <a href="#">Pinjamanku</a>
    </li>
    <li class="BottomBar-menuItem">
      <a href="#">Riwayat</a>
    </li>
    <li class="BottomBar-menuItem is-active">
      <a href="#">Lainnya</a>
    </li>
  </ul>
</div>
```
