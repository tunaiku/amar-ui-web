## Usage

<div class="BgColor-neutral-20 Padding-2xlarge">
  <div class="Snackbar-dialog">
    <div class="Snackbar-text">A single line message.</div>
    <div class="Snackbar-actions">
      <button class="Snackbar-action">Action</button>
    </div>
  </div>
</div>


## Variants

#### Longer Action Button

To make room for a longer action button, append `Snackbar-dialog--long` after `Snackbar-dialog`.

```html preview=true
<div class="Snackbar-dialog Snackbar-dialog--long">
  <div class="Snackbar-text">
    A single line message.
  </div>
  <div class="Snackbar-actions">
    <button class="Snackbar-action">Longer Action Button</button>
  </div>
</div>
```

#### Elevated

To give the snackbar an elevated effect, append `Snackbar-dialog--elevated` modifier class.

```html preview=true
<div class="Snackbar-dialog Snackbar-dialog--elevated">
  <div class="Snackbar-text">
    A single line message.
  </div>
  <div class="Snackbar-actions">
    <button class="Snackbar-action">Action</button>
  </div>
</div>
```
