from the user. It covers the whole front of the primary content until dismissed or confirmed."
---

import DialogExample from 'examples/dialog-example';

## Usage

```html preview=true center=true bgColor=neutral.10
<div class="Dialog-box">
  <div class="Dialog-block">
    <div class="Heading-5 FontWeight-bold PaddingBottom">Dialog Title</div>
    <p class="Color-gray-50">Description text here.</p>
  </div>
  <div class="Dialog-block Flex JustifyContent-end">
    <button class="Button Button--text MarginRight-2xsmall Button--primary">
      Decline
    </button>
    <button class="Button Button--primary">Submit</button>
  </div>
</div>
```
