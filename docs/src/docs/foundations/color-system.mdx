## Color shades

Each color theme has different shades generated by mixing both white and black color. These shades
are divided by **shade levels**. By default, the shade levels ranges from 10 (lighter) to 90
(darker), where 50 is the base shade for each color theme.

## Customizing themes

Customizing the default themes can be done simply by adjusting the key-value pair inside
`$amb-color-theme-map` variable.

Below is the default key-value pairs that make up the default themes of Amar UI Web.

```scss
/// Map variables for color themes.
///
$amb-color-theme-map: () !default;
$amb-color-theme-map: map-merge(
  (
    'primary': $amb-primary,
    'secondary': $amb-secondary,
    'tertiary': $amb-tertiary,
    'success': $amb-success,
    'warning': $amb-warning,
    'danger': $amb-danger,
    'info': $amb-info,
    'neutral': $amb-neutral,
    'light': $amb-neutral-30,
    'lighter': $amb-neutral-10,
    'dark': $amb-neutral-80,
    'darker': $amb-neutral-90,
  ),
  $amb-color-theme-map
);
```

To add a new theme, you can simply add a new key-value pair inside the map variable. For example, if
we're to add a new theme called `red`:

```scss
// In your override file
$amb-color-theme-map: (
  'red': <color-value>,
);
```

This enables `red` theme to be used anywhere that utilizes the theme module.

```scss
.Color--red {}
.BgColor--red {}
.Button--red {}
...
```
