## Color shades

By utilizing tinter and shader utility, we can create multiple shades of a color
theme that fit for multiple purposes by mixing a color theme with either white
and black. Each color theme has a range of shades from `0 (lighter)` to
`100 (darker)`. Shade value of `50` is comparable to base color value.

Refer to color utilities documentation to see how you can manipulate shade of
color themes easily.

## Customizing Color Themes

Customizing color themes is a simple and straightforward process of overriding
the variables used to populate the color themes map variable in
`$amb-color-theme-map`.

```scss
@import '@amar-ui-web/color';
```

## Functions

Functions are also exposed for those who want more flexibility in color
manipulation.

#### `amb-color-theme`

Returns a color hex value based on the passed color theme included inside
`$amb-color-theme-map` map variable. Accepts one argument:

- `$color-theme | String`

```scss
@import '@amar-ui-web/color/scss/functions';

// Input
.Example {
  color: amb-color-theme('primary');
}
```
