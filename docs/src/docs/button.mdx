## Usage

Add the `.Button` class in a `<button>` or `<a>` tag to apply the standard button styling.

```html preview=true fullWidth=true
<button class="Button">Default button</button>
```

A little note for accessibility; adding `.Button` class to `<a>` is completely possible, but
remember to always put `role="button"` attribute. This will make sure screen readers can recognize
`<a>` element as a clickable element.

## Themes

Append `Button--<theme>` class to turn your class into a themed button. Themed button has filled
background color with the selected color theme of choice.

```html preview=true fullWidth=true
<button class="Button Button--primary">Primary</button>
<button class="Button Button--secondary">Secondary</button>
<button class="Button Button--success">Success</button>
<button class="Button Button--info">Info</button>
<button class="Button Button--warning">Warning</button>
<button class="Button Button--danger">Danger</button>
```

## Variants

You can modify a button with a handful of **variants**. Note that variants can only work in a
**themed button**.

```html preview=true fullWidth=true
<button class="Button Button--primary">Button</button>
<button class="Button Button--primary Button--outlined">Button</button>
<button class="Button Button--primary Button--text">Button</button>
<button class="Button Button--primary Button--elevated">Button</button>
```

#### Outlined button

Outlined button has a lower emphasis than a themed button and is usually used for secondary actions.

To apply an outlined button, append `.Button--outlined` alongside `Button` and `Button--<theme>`.

```html preview=true bgColor=neutral.10 fullWidth=true
<button class="Button Button--outlined Button--primary">Primary</button>
<button class="Button Button--outlined Button--secondary">Secondary</button>
<button class="Button Button--outlined Button--tertiary">Tertiary</button>
<button class="Button Button--outlined Button--success">Success</button>
<button class="Button Button--outlined Button--info">Info</button>
<button class="Button Button--outlined Button--warning">Warning</button>
<button class="Button Button--outlined Button--danger">Danger</button>
```

#### Text button

**Text button** has no border and background fill, leaving only a clickable text. It has the
**lowest emphasis** than any other variants.

To apply a text button, append `.Button--text` alongside `Button` and `Button--<theme>`.

```html preview=true fullWidth=true
<button class="Button Button--text Button--primary">Primary</button>
<button class="Button Button--text Button--secondary">Secondary</button>
<button class="Button Button--text Button--tertiary">Tertiary</button>
<button class="Button Button--text Button--success">Success</button>
<button class="Button Button--text Button--info">Info</button>
<button class="Button Button--text Button--warning">Warning</button>
<button class="Button Button--text Button--danger">Danger</button>
```

#### Elevated button

Elevated button has a custom effect to make your button appear floated, similar to Material Design's
button.

```html preview=true fullWidth=true
<button class="Button Button--elevated Button--primary">Primary</button>
<button class="Button Button--elevated Button--secondary">Secondary</button>
<button class="Button Button--elevated Button--tertiary">Tertiary</button>
<button class="Button Button--elevated Button--success">Success</button>
<button class="Button Button--elevated Button--info">Info</button>
<button class="Button Button--elevated Button--warning">Warning</button>
<button class="Button Button--elevated Button--danger">Danger</button>
```

## Modifiers

Modifiers differ than variants in that they are freely mix-and-matched with each other to form
different button entities.

#### Sizes

Append `Button--<size>` modifier class to change the size of a button. Out of the box, there are
three different sizes you can choose from.

```html preview=true fullWidth=true
<button class="Button Button--primary Button--small">Small</button>
<button class="Button Button--primary">Base</button>
<button class="Button Button--primary Button--large">Large</button>
```

#### Rounded

Append `Button--rounded` modifier to turn buttons into a pill-like shape.

```html preview=true fullWidth=true
<button class="Button Button--primary Button--rounded">A rounded button</button>
<button class="Button Button--secondary Button--outlined Button--rounded">A rounded button</button>
```

#### Full Width

Append the `Button--fullWidth` modifier to increase a button's width to expand over its parent's
width.

```html preview=true fullWidth=true
<button className="Button Button--primary Button--fullWidth MarginBottom">Primary Button</button>
<button className="Button Button--secondary Button--outlined Button--fullWidth MarginBottom">
  Outlined Button
</button>
```

#### Disabled

Turns a button into disabled state which makes it **noninteractive**.

```html preview=true fullWidth=true
<button className="Button Button--primary" disabled>Disabled button</button>
<a className="Button Button--primary is-disabled">Disabled button</a>
```
