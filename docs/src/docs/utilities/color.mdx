## Text Color

Use `.Color-<theme>` class to change the **text color** of an element.

```html preview=true
<div class="Color-primary Padding-small">Primary Color</div>
<div class="Color-secondary Padding-small">Secondary Color</div>
<div class="Color-tertiary Padding-small">Tertiary Color</div>
<div class="Color-danger Padding-small">Danger Color</div>
<div class="Color-warning Padding-small">Warning Color</div>
<div class="Color-info Padding-small">Info Color</div>
<div class="Color-light Padding-small">Light Color</div>
<div class="Color-dark Padding-small">Dark Color</div>
```

## Background color

Use `.BgColor-<theme>` class to change the **background color** of an element.

```html preview=true
<div class="BgColor-primary Padding-small">Primary Color</div>
<div class="BgColor-secondary Padding-small">Secondary Color</div>
<div class="BgColor-tertiary Padding-small">Tertiary Color</div>

<div class="BgColor-info Padding-small">Info Color</div>
<div class="BgColor-light Padding-small">Light Color</div>
<div class="BgColor-dark Padding-small">Dark Color</div>
```

## Color shades

To use a different shade of a color theme for text or background color in an element, simply add
`*-<shadeLevel>`. By default, you can choose shade level ranging from 10 to 90.

#### Text color shades

```html preview=true
<div class="Color-primary Padding-small">Primary color (base)</div>
<div class="Color-primary-10 Padding-small">Primary 10</div>
<div class="Color-primary-20 Padding-small">Primary 20</div>
<div class="Color-primary-30 Padding-small">Primary 30</div>
<div class="Color-primary-40 Padding-small">Primary 40</div>
<div class="Color-primary-50 Padding-small">Primary 50 (same as base)</div>
<div class="Color-primary-60 Padding-small">Primary 60</div>
<div class="Color-primary-70 Padding-small">Primary 70</div>
<div class="Color-primary-80 Padding-small">Primary 80</div>
<div class="Color-primary-90 Padding-small">Primary 90</div>
```
