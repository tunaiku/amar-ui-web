## Color shades

To apply shade of a particular color, simply use `Color-<name>-<shade>` where shade is between 10 to
90 in an increment of 10 (10, 20, ..., 90).

To learn more about shades, check out the [Color System](/foundations/color-system) page.

```html preview=true
<div class="Color-primary-30">Primary 30</div>
<div class="Color-success-70">Success 70</div>
<div class="Color-neutral-90">Neutral 90</div>
```
