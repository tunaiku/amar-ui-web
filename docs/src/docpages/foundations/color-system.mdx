## Shades

Each color type be it a brand color, or an accent color **except neutral color** have shades that
are automatically generated using a built-in tinter and shader function. These functions
respectively mixes a color with a mixture of pure black (`#000000`) and white (`#ffffff`).

Each shade is represented in an increment of 10 from 10 to 90. The lightest shade is represented by
10, and 90 is for the darkest shade. The default shade is 50.
