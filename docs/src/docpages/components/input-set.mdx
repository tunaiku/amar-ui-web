## Usage

You can append `.InputSet` class into form control container and then append one of 2 modifiers:

1. `.InputSet--hasIconLeft`
2. and/or `.InputSet--hasIconLeft`

You also need to add a modifier on the icon container:

1. `InputSet-icon InputSet-icon--left` if `.InputSet--hasIconLeft` is used
2. `InputSet-icon InputSet-icon--right` if `.InputSet--hasIconRight` is used
