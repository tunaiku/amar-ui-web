---
title: 'Tips'
description:
  'Displays temporary information that happen in the background which can be dismissed by the user.'
path: '/components/tips'
---

## Usage

```html preview=true size=compact
<div class="Tips">
  <p>What tips should the user know?</p>
</div>
```

## With Link

Simply add a link element as a child inside the .Tips wrapper class to add an actionable
call-to-action as needed.

```html preview=true size=compact
<div class="Tips">
  <p>What tips should the user know?</p>
  <a class="Tips-link" href="#">What should the user get?</a>
</div>
```
