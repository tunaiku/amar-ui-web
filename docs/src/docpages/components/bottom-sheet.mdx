  "A bottom sheet is a type of modal that is attached to the bottom of the viewport. Much like
  dialogs, it covers the front of the primary content until dismissed. Use the bottom sheet to
  divert user's attention to important information."
---

## Usage

sheet wrapper. Then add `.BottomSheet-dialog` to create a container to place the content you want to
put.

## Fixed

Append `.BottomSheet--fixed` class to make it stick to the bottom of the screen.

```html
<div class="BottomSheet BottomSheet--fixed">
  <div class="BottomSheet-dialog">...</div>
</div>
```
