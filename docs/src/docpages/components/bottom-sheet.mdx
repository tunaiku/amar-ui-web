## Fixed

By default, `.BottomSheet` component will be placed according to the parent's location. Since bottom
sheet primary intention is to divert user's attention, we need a way to attach it to the viewport.
As its name suggests, bottom sheet should be attached at the bottom of the screen. Simply append
`.BottomSheet--fixed` to do this.

```html
<div class="BottomSheet BottomSheet--fixed">
  <div class="BottomSheet-dialog">...</div>
</div>
```
