## Usage

Field is composed into several parts; `Field`, `Field-label`, `Field-control`, and the optional
`Field-helper`.

```html preview=true size=compact

      <input type="text" placeholder="e.g John Appleseed" class="Input" />

  </div>
  <div class="Field MarginBottom">
    <label class="Field-label">Password</label>
```

#### `Field-label`

`Field-label` is the class to put in a label associating the field. It's should be placed as the
first child of a `Field` class.

#### `Field-control`

`Field-control` is the wrapper class where you can put the preferred form control like `Input` and
`Select`. You can even use `InputSet` if you have a control with icons.

#### `Field-helper`

`Field-helper` is an optional class to put in helper text that should be put beneath a
`.Field-control`. This can be used to display error messages or assistive text.

## Applying themes

At the moment, you apply theme modifiers or color utilities directly to the elements in a field
component as needed.

In the example below, we apply `.Input--danger` to the input element and `Color-danger` to the label
to achieve an invalid field.

```html preview=true size=compact
<div class="Field">
  <div class="Field-label Color-danger">Form label</div>
  <div class="Field-control">
    <input type="text" class="Input Input--danger" placeholder="Some placeholder" />
  </div>
</div>
```
