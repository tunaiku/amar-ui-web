## Usage

Append class `.IconCard` consists of several elements that you can mix and match according to your needs.

- `.IconCard`: Serves as the primary wrapper class for the other parts.
- `.IconCard-image`: Inside this class you can bind `img` element.
- `.IconCard-title`: Contains the title and subtitle text of the card.
- `.IconCard-body`: Contains the primary content of the card.
- `.IconCard-action`: Is wrapper that contains to call-to-action elements of the card (e.g button, link, etc.).
- `.IconCard-chevron`: Contains the icon (chevron). This part you can take out and change it with other icon or button.

## Without Image

```html preview=true bgColor=lighter size=compact
<div class="IconCard">
  <div class="IconCard-image"></div>
  <div class="IconCard-body">
    <div class="IconCard-title">
      <h5 class="Typography-bold16">Title Card here</h5>
    </div>
    <div class="IconCard-content">
      <p class="Typography-regular14 Color-neutral-80">
        Description text here. Contains various supporting text or content.
      </p>
    </div>
  </div>
  <div class="IconCard-action">
    <div class="IconCard-chevron"></div>
  </div>
</div>
```
