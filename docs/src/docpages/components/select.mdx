## Usage

To create a select component, simply create an outer element with the class `.Select` followed by
the `<select>` element.

```html preview=true size=compact
<div class="Select">
  <select>
    <option>Jakarta</option>
    <option>Bandung</option>
    <option>Surabaya</option>
  </select>
</div>
```
