## Usage

Spacing can be implemented either around the element or only on specific sides you choose.
Simply append which side you want to give spacing onto. The syntax to implement a spacing is
`<Margin|Padding><Top|Right|Bottom|Left>-<size>`.

- `<Margin|Padding>` adds spacing, either margin or padding, around an element.
- `<Margin|Padding>Top` adds spacing on the **top** side of an element.
- `<Margin|Padding>Right` adds spacing on the **right** side of an element.
- `<Margin|Padding>Bottom` adds spacing on the **bottom** side of an element.
- `<Margin|Padding>Left` adds spacing on the **left** side of an element.

```html
<div class="Margin">...</div>
<div class="Padding">...</div>
<div class="MarginTop">...</div>
<div class="PaddingBottom">...</div>
```

## Spacing sizes

Spacing comes with multiple sizes out of the box with `1rem` or `16px` computed being the base
size of spacing. To apply a different spacing size to an element, simply append `*-<size>` class.

- `2xsmall`:`0.25rem` or `4px`.
- `small`: `0.5rem` or `8px`.
- `base`: `1rem` or `16px`.
- `medium`: `1.5rem` or `24px`
- `large`: `2.25rem` or `32px`

```html preview=true
<div class="Padding-2xsmall BgColor-neutral MarginBottom"><code>4px</code></div>
<div class="Padding-xsmall BgColor-neutral MarginBottom"><code>8px</code></div>
<div class="Padding-small BgColor-neutral MarginBottom"><code>12px</code></div>
<div class="Padding-base BgColor-neutral MarginBottom"><code>16px</code></div>
<div class="Padding-medium BgColor-neutral MarginBottom"><code>24px</code></div>
<div class="Padding-large BgColor-neutral"><code>32px</code></div>
```

## Spacing variables

These are the variables that make up our spacing classes and functions that can be further seen
in the spacing utility page.

In Amar UI Web, we use the base value of `1rem` or `16px` as the default spacing value.

```scss
$amb-spacing-map: () !default;
$amb-spacing-map: map-merge(
  (
    '2xsmall': 0.25rem,
    'xsmall': 0.5rem,
    'small': 0.75rem,
    'base': 1rem,
    'medium': 1.5rem,
    'large': 2.25rem,
    'xlarge': 3rem,
    '2xlarge': 4.5rem,
  ),
  $amb-spacing-map
);

$amb-base-spacing: amb-spacing('base') !default;
```
