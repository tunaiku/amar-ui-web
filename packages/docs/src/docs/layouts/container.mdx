## Usage

To implement a container, simply add the `.Container` class.

```html
<div class="Container">
  <!-- ... -->
</div>
```

## Example

In the holy grail layout illustration below, you can see that the body section utilizes `Container`
class to horizontally center the contents inside.

<div className="CodeBlock">
  <div className="CodeBlock-display">
    <header className="BgColor-secondary Padding TextAlign-center">Header Area</header>
    <main className="BgColor-neutral-10 Flex Flex-column" style="height: 20em">
      <div
        className="Container BgColor-secondary-20 Flex AlignItems-center JustifyContent-center FlexGrow-1"
        style="width: 450px">
        Container Area
      </div>
    </main>
    <footer className="BgColor-neutral-90 Padding TextAlign-center">Footer Area</footer>
  </div>
</div>

```html
<header class="Header">Header Area</header>
<main class="Content">
  <div class="Container">Container Area</div>
</main>
<footer class="Footer">Header Area</footer>
```
