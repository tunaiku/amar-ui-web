## Usage

To start implementing a grid layout, first transform an element into a grid wrapper by
appending the `.Grids` class (notice the plural declaration). Then, you can start assembling
grid items as direct children of that wrapper by appending `.Grid` class.

- `.Grids`: Turns an element into a grid wrapper.
- `.Grid`: Turns an element into a grid item.

Each grid item is separated by a gutter of `1rem` that translates to `16px`.

```html preview=true
<div class="Grids">
  <div class="Grid">
    <div class="BgColor-secondary-50 Padding">A grid item</div>
  </div>
  <div class="Grid">
    <div class="BgColor-secondary-50 Padding">A grid item</div>
  </div>
  <div class="Grid">
    <div class="BgColor-secondary-50 Padding">A grid item</div>
  </div>
</div>
```

## Responsive Grids

You can achieve specific grids on certain breakpoints by adding `Grid-<breakpoint>-<value>`.

```html preview=true
<div class="Grids MarginBottom">
  <div class="Grid Grid-12 Grid-sm-6 Grid-md-4">
    <div class="BgColor-secondary Padding">A grid item</div>
  </div>
  <div class="Grid Grid-12 Grid-sm-6 Grid-md-4">
    <div class="BgColor-secondary Padding">A grid item</div>
  </div>
  <div class="Grid Grid-12 Grid-sm-6 Grid-md-4">
    <div class="BgColor-secondary Padding">A grid item</div>
  </div>
</div>
<div class="Grids">
  <div class="Grid Grid-12 Grid-sm-6 Grid-md-3">
    <div class="BgColor-secondary Padding">A grid item</div>
  </div>
  <div class="Grid Grid-12 Grid-sm-6 Grid-md-3">
    <div class="BgColor-secondary Padding">A grid item</div>
  </div>
  <div class="Grid Grid-12 Grid-sm-6 Grid-md-3">
    <div class="BgColor-secondary Padding">A grid item</div>
  </div>
  <div class="Grid Grid-12 Grid-sm-6 Grid-md-3">
    <div class="BgColor-secondary Padding">A grid item</div>
  </div>
</div>
```

## Nested Grid

You can nest grids indefinitely as needed by simply adding another `.Grids` class inside of
a `.Grid`.

```html preview=true
<div class="Grids">
  <div class="Grid Grid-6">
    <div class="BgColor-secondary-30 Padding">
      <div class="Grids">
        <div class="Grid">
          <div class="BgColor-secondary Padding">A nested grid</div>
        </div>
        <div class="Grid">
          <div class="BgColor-secondary Padding">A nested grid</div>
        </div>
      </div>
    </div>
  </div>
  <div class="Grid Grid-6">
    <div class="BgColor-secondary Padding">A grid item</div>
  </div>
</div>
```
